# forvoztools 🔊

[![License: GPL v3](https://img.shields.io/badge/License-GPLv3-blue.svg)](https://www.gnu.org/licenses/gpl-3.0)

**forvoztools** (Forensic Voice Tools) es un paquete de R diseñado para el análisis forense de voz, específicamente para el cálculo y visualización de LLR (Log-Likelihood Ratio).

## 📦 Instalación

### Desde GitHub (recomendado)

```r
# Instalar remotes si no lo tienes
if (!require("remotes")) install.packages("remotes")

# Instalar forvoztools
remotes::install_github("TU_USUARIO/forvoztools")
